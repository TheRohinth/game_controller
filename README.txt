This is a game controller developed using python.
run the 'main.py' file.
A small window will display the frames from your local camera.

Controls :-

1) Forward/Up - Thumbs up
2) Backwarrd/Down - Thumbs folded
3) Turn Left - Right hand to the top and left hand to the bottom
4) Turn Left - Left hand to the top and Right hand to the bottom
